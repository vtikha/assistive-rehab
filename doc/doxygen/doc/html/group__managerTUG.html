Description

Parameters

  • –module_name: Name of the module. All the open ports will be tagged with the prefix /module_name.
  • –period: Periodicity of the module.
  • –speak-file: Configuration file name for the speak.
  • –finish-line-thresh: Threshold on the distance between foot and finish line [m].
  • –standing-thresh: Threshold on the speed of the shoulder center height [m/s].
  • –starting-pose: Robot initial pose (x,y,theta) wrt world frame when starting the TUG.

Input Ports

Output Ports

  • /managerTUG/attention:rpc [rpc] [default carrier:]: Sends commands to attentionManager to redirect the robot's gaze toward the engaged user.
  • /managerTUG/analyzer:rpc [rpc] [default carrier:]: Sends commands to motionAnalyzer for starting/stopping the real-time motion analysis.
  • /managerTUG/speech:rpc [rpc] [default carrier:]: Sends commands to iSpeak to wait until the sentence is spoken.
  • /managerTUG/speech:o [Bottle] [default carrier:]: Streams out the verbal interaction. To be connected to iSpeak.

Services

  • /managerTUG/cmd:rpc [rpc-server]: service port . This service is described in managerTUG_IDL (idl.thrift)
